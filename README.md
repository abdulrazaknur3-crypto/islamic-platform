# المنصة الإسلامية — Islamic Platform — ኢስላሚያ መድረኽ

منصة دعوية توعوية تثقيفية للمجتمع الإرتري بثلاث لغات (العربية، التجرينية، الإنجليزية).

## التقنيات
- Next.js 14 (App Router)
- next-intl (i18n مع تبديل RTL/LTR تلقائي)
- Tailwind CSS
- TypeScript

## الهوية اللونية (البحر والرمل)
- Primary: `#0E5C63`
- Secondary: `#17858E`
- Sand: `#D9B98A`
- Cream: `#F7F3EC`

## التشغيل محليًا
```bash
npm install
npm run dev
```
ثم افتح http://localhost:3000 (يعيد التوجيه تلقائيًا إلى /ar).

## اللغات
- `/ar` العربية (RTL)
- `/en` English (LTR)
- `/ti` ትግርኛ (LTR)

ملفات الترجمة في مجلد `messages/`.

**تنبيه:** الترجمة التجرينية أولية وتحتاج مراجعة ناطق أصلي قبل الإطلاق.
